# 🧠 NTI NLP Track

A collection of tasks, assignments, and projects completed during the
**Natural Language Processing (NLP) Track at the National Telecommunication Institute (NTI).**

This repository documents my practical learning journey in NLP, starting from
text preprocessing and vectorization to deep learning and advanced NLP concepts.

---

## 📚 About This Repository

This repository contains the practical work, tasks, and projects I completed
throughout the NTI NLP Track.

The content covers both **traditional NLP techniques** and **Deep Learning approaches**
for Natural Language Processing.

### Topics Covered

- 🧹 Text Preprocessing
- 🔢 Text Vectorization
- 📐 Document Similarity
- 🔄 RNN & BiRNN

## 🧩 Sessions & Topics

### 🧹 Session 01 — Text Preprocessing

Exploring the fundamentals of preparing raw text for NLP tasks.

Topics include:

* Text Cleaning
* Regular Expressions
* Tokenization
* Stopword Removal
* Stemming
* Lemmatization
* Text Normalization

📓 Notebook:

`01_text_preprocessing.ipynb`

---

### 🔢 Session 02 — Text Vectorization

Learning how to represent text as numerical features that machine learning
models can process.

Topics include:

* Bag of Words
* TF-IDF
* Word Embeddings
* Word2Vec

📓 Notebook:

`02_text_vectorization.ipynb`

---

### 📐 Session 03 — Document Similarity

Exploring methods for measuring similarity between text documents.

Topics include:

* TF-IDF
* Cosine Similarity
* Document Representation
* Text Similarity Analysis

📓 Notebook:

`03_document_similarity.ipynb`

📓 Project:

`03_document_similarity_project.ipynb`

---

### 🔄 Session 04 — RNN & BiRNN

Exploring Recurrent Neural Networks for processing sequential data and
comparing unidirectional and bidirectional architectures.

Topics include:

* Recurrent Neural Networks (RNN)
* Bidirectional RNN (BiRNN)
* Sequence Modeling
* Model Comparison

📓 Notebook:

`04_rnn_vs_birnn.ipynb`

🐍 Python Application:

`04_rnn_vs_birnn_app.py`
